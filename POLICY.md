## 개인정보처리방침

**최종 수정일: 2025년 12월 01일**

본 개인정보처리방침은 Safer Dreamcord (이하 "본 확장 프로그램")이 사용자의 개인정보를 어떻게 수집, 이용, 공유하는지 설명합니다.

### 1. 수집하는 개인정보 항목

본 확장 프로그램은 다음의 정보를 수집합니다:

- **드림핵(Dreamhack) 계정 ID**: 사용자 식별 목적
- **풀이 완료한 문제 ID**: 문제 풀이 현황 추적 목적
- **사용자 지정 웹훅 URL**: 알림 전송 목적 (사용자가 직접 설정)

### 2. 개인정보 수집 및 이용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

- 사용자가 설정한 웹훅 URL로 문제 풀이 완료 알림 전송
- 확장 프로그램의 핵심 기능 제공

### 3. 개인정보의 제3자 제공

본 확장 프로그램은 수집된 정보를 **사용자가 직접 지정한 웹훅 URL로만** 전송합니다. 개발자는 사용자의 개인정보를 별도로 수집하거나 저장하지 않으며, 제3자에게 판매, 공유, 임대하지 않습니다.

### 4. 개인정보의 보관

- 모든 데이터는 **사용자의 로컬 브라우저 저장소(Chrome Storage)**에만 저장됩니다.
- 개발자 서버로 데이터가 전송되거나 저장되지 않습니다.
- 확장 프로그램 삭제 시 모든 저장된 데이터가 자동으로 삭제됩니다.

### 5. 사용자의 권리

사용자는 언제든지 다음을 수행할 수 있습니다:

- 확장 프로그램 설정에서 저장된 정보 확인 및 수정
- 웹훅 URL 변경 또는 삭제
- 확장 프로그램 삭제를 통한 모든 데이터 삭제

### 6. 데이터 보안

본 확장 프로그램은 사용자 데이터를 보호하기 위해 다음 조치를 취합니다:

- 외부 서버로의 불필요한 데이터 전송 금지
- 사용자가 명시적으로 설정한 웹훅으로만 데이터 전송
- HTTPS를 통한 안전한 데이터 전송 권장

### 7. 개인정보처리방침의 변경

본 방침이 변경될 경우, 확장 프로그램 업데이트 또는 관련 페이지를 통해 사전 고지합니다.
